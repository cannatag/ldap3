Welcome to ldap3's documentation
=======================================

ldap3 is a pure Python LDAP 3 client library strictly conforming to RFC4511 released under the LGPL v3 open source license.
RFC4511 is the current LDAP specification (June 2006) from IETF and obsoletes the previous LDAP RFCs 2251, 2830, 3771 (December 1997)

ldap3 can be used on Python 3 and Python 2 (starting from python 2.6). It works on PyPy and PyPy3.
